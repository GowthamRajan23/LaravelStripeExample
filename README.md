# Laravel Stripe Example

## Introduction

This package shows the integration of Stripe payment gateway using Laravel 8.

## Versions Used
- Laravel 8.83
- Laravel Grid View 
- Bootstrap 4 for styling
- JQuery
- PHP 7.4
- Composer
- Stripe Payment Gateway
